# 🧠 Customer Segmentation System

A Machine Learning project that segments customers using K-Means clustering and PCA for dimensionality reduction.

## 🚀 Features
- Customer clustering
- PCA-based dimensionality reduction
- Interactive Streamlit dashboard
- Real-time cluster prediction

## 🛠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas, NumPy, Matplotlib

## 📊 Pipeline
Raw Data → Scaling → PCA → KMeans
